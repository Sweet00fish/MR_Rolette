# MR_Rolette
Marvel Raivalsのキャラクターをルーレットできるアプリです。</br>

<img width="578" height="663" alt="スクリーンショット 2026-02-12 112511" src="https://github.com/user-attachments/assets/efff3517-57b1-4e78-8791-390d7c0eb8ed" />

旧バージョンを使用していた場合は、フォルダごと上書きして更新してください。</br>
更新すると保存されていたチェックボックスのデータは初期化されます。ご留意ください。</br></br>

**中身**
- Roulette.exe : 実行ファイルです。</br>
- readme.txt : readmeです。</br>
- License.txt : ライセンスファイルです。再配布等の場合にお使いください。</br>

**使い方**
- スタート : ルーレットを始めます。</br>
- ストップ : ルーレットを止めます。ゆっくり止まります。</br>
- チェックリスト : チェックが入っているヒーローだけがルーレットの対象です。</br>
- 全チェック : 各ロールの全チェックON/OFFができます。</br></br>

*※アプリを閉じると、閉じた際のチェック項目が自動保存されます。</br>
※実行ファイルを何度も実行できるため、複数ウィンドウを同時に開くことが可能です。</br>
特にバグは発生しないと思われますがもし不具合がありましたら報告お願いします。</br></br>*
